Static pages and sources:

Open Targets information

Diseases are described by EFO codes. Rare diseases on OT use Orphanet codes; these can also be used.

Network source + information - STRING, OmniPath

Networks have been filtered to remove low confidence edges, then the largest connected component extracted.

Algorithm information - HotNet2

All subnetworks are extracted using the HotNet2 algorithm. A beta-value is calculated for each source network on setup, and delta-values are automatically selected as part of the subnetwork detection process.
